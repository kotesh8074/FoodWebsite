# 🍽️ Food Website

A modern and responsive restaurant website developed using HTML, CSS, JavaScript, Bootstrap, Swiper.js, and jQuery.

## 🚀 Features

* Responsive Design
* Modern User Interface
* Interactive Food Menu
* Food Gallery
* Chef Team Section
* Customer Testimonials
* Restaurant Booking Section
* Blog Section
* Newsletter Subscription
* Smooth Scrolling Effects
* Mobile Friendly Layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* jQuery
* Swiper.js
* Fancybox
* GSAP

## 🔧 Installation

1. Clone the repository

```bash
https://github.com/kotesh8074/FoodWebsite.git
```

2. Open the project folder

3. Run `index.html` in your browser
